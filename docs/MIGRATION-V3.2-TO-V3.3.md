# Migration from EDIS 3.2.0 to 3.3.0

1. Back up the site and database.
2. Deactivate 3.2.0 without uninstalling it.
3. Install and activate 3.3.0.
4. Run Diagnostics and a new export.

Version 3.3 added explicit export scopes, single/multiple-document selection, post-processing dependency execution, source coverage, diagnostics aggregation, schema routing, document/element indexes and page-row actions. Incomplete 3.2 jobs must be recreated.
